El Data Lake de Reservamos SaaS permite a nuestros clientes recolectar datos de sus usuarios para posteriormente consumir servicios potenciados por IA que aportan valor a sus productos.

### Los datos son la base

Para que nuestros servicios puedan aportar valor, es necesario recolectar datos. Esto se logra a través de nuestro SDK, que puede ser integrado en cualquier producto digital.

Para hacer uso de nuestros servicios será necesario contar con un cuenta de Mixpanel y Fingerprint. Los tokens de estos dos servicios son requeridos para configurar el SDK.
